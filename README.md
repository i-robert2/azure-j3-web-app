# Azure J3 — Static Web App + custom domain + TLS

Ship a static site on **Azure Static Web Apps** (Free SKU), deploy it via **GitHub Actions**, bind a **custom domain** with a free auto-renewing **TLS** certificate, manage **DNS in Azure**, instrument **Application Insights**, and gate pull requests behind a **Lighthouse CI** performance bar — with hardened security headers.

> Work in progress. Built as a hands-on learning project.
